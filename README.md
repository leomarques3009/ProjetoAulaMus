# Projeto Aula de Música
## Objetivos Gerais
<p>Esse projeto busca facilitar o controle de alunos de um professor, com o registro dos alunos, aulas e contratos.</p>
<p>Futuramente o projeto busca implementar filtros de alunos, por nome, cpf ou RG, filtros de aulas e contratos por data, exibir um "calendário" semanal com os horários de aula ativos.</p>
<p>Como desafio, e principal obejtivo, o projeto deve criar um documento do contrato e gerar como PDF</p>
